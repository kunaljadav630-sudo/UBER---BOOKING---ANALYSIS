# 🚕 Uber Booking Analysis Dashboard

## 📌 Project Overview
This project analyzes Uber booking data using Excel, SQL, and Power BI to identify booking trends, customer behavior, revenue insights, and vehicle performance.

## 🛠️ Tools Used
- Excel
- SQL
- Power BI
- DAX
- Power Query

## 📊 Key KPIs
- Total Bookings
- Successful Bookings
- Booking Value
- Average Booking Value
- Average Distance
- Cancellation Rate
- Customer Ratings
- Driver Ratings

## 📈 Dashboard Features
- KPI Cards
- Bar Charts
- Line Charts
- Matrix Tables
- Slicers
- Interactive Filters

## 📂 Files Included
- Dataset (.csv)
- SQL Queries (.sql)
- Dashboard Screenshots
- Power BI Dashboard (.pbix)
